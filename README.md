## Technologies

-   Node.js + Express.js
-   MongoDB
-   React.js

## Instructions to run

-   Pre-requisites:

    -   Node.js
    -   NPM

-   Directions to setup/install

```bash
$ git clone https://github.com/dsckiet/uddeshhya_backend
$ cd uddeshhya_backend
$ touch .env
$ cp .env.example .env
# Set values in .env
$ npm i
```

-   Directions to execute

```bash
$ npm run start:dev
```
